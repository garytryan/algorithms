# Multiply Two matrices

Given tow n*n matrices return the product.

- in O(1) space
- in O(n^3) time complexity

### Example

```
    matrixA = [
        [1, 2],
        [3, 4]
    ]

    matrixB = [
        [5, 6],
        [7, 8]
    ]

    product = multiplyMatrix(matrixA, matrixB)

    product == [
        [19, 22],
        [43, 50]
    ]
```
